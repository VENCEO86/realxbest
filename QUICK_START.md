# 빠른 시작 가이드

## 서버 실행

개발 서버가 포트 3001에서 실행 중입니다.

### 브라우저에서 확인

1. **메인 페이지**: http://localhost:3001
2. **관리자 페이지 (광고 관리)**: http://localhost:3001/admin/ads

## 현재 상태

- ✅ 프로젝트 구조 완성
- ✅ 기본 페이지 및 컴포넌트 구현
- ⚠️ 데이터베이스 설정 필요 (선택사항)

## 데이터베이스 없이도 확인 가능

현재 데이터베이스가 설정되지 않아도 UI는 확인할 수 있습니다:
- 페이지 레이아웃
- 필터 UI
- 광고 관리자 UI

## 데이터베이스 설정 (데이터 표시를 위해)

1. `.env` 파일 생성:
```env
DATABASE_URL="postgresql://user:password@localhost:5432/korxyoutube?schema=public"
```

2. 데이터베이스 마이그레이션:
```bash
npm run db:generate
npm run db:push
```

3. 서버 재시작

## 문제 해결

### 서버가 시작되지 않는 경우
- 포트 3001이 사용 중인지 확인
- `npm install` 실행 확인
- 터미널에서 오류 메시지 확인

### 페이지가 표시되지 않는 경우
- 브라우저 캐시 삭제
- 하드 리프레시 (Ctrl+Shift+R)
- 서버 로그 확인
