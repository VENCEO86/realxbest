   파일: d:\sscom\lib\common.lib.php
   
   변경 전:
   ```php
   mysqli_set_charset($link, 'utf8');
   ```
   
   변경 후:
   ```php
   mysqli_set_charset($link, 'utf8mb4');
   mysqli_query($link, "SET NAMES utf8mb4");
   mysqli_query($link, "SET CHARACTER SET utf8mb4");
   ```

2. 공통 문자셋 설정 변경
   파일: d:\sscom\common.php (148번 줄)

   ```php

   ```
   
   변경 후:
   ```php
   sql_set_charset('utf8mb4', $connect_db);
   ```

UTF-8 vs UTF-8MB4:
  - UTF-8: 최대 3바이트 문자 지원 (기본 한글, 영문)
  - UTF-8MB4: 최대 4바이트 문자 지원 (이모지 포함)
  - MySQL 8.0부터 utf8mb4 권장

character_set 종류:
  - character_set_client: 클라이언트가 보내는 쿼리의 문자셋
  - character_set_connection: 쿼리 처리 시 사용하는 문자셋
  - character_set_results: 결과를 클라이언트에 보낼 때 사용하는 문자셋

SET NAMES utf8mb4:
  - 위 3가지를 한번에 설정하는 명령어
  - character_set_client = utf8mb4
  - character_set_connection = utf8mb4
  - character_set_results = utf8mb4
