# 🎓 Render 환경 변수 설정 - 초등학생도 이해하는 가이드

## 🎯 목표
Render에 우리 웹사이트가 작동하도록 필요한 정보를 알려주는 것입니다!

---

## 📚 배경 설명 (초등학생 이해도)

### 🤔 환경 변수가 뭐예요?

**비유로 설명하면:**
- 우리가 학교에 가려면 "학교 이름", "교실 번호", "선생님 이름"을 알아야 해요
- 웹사이트도 마찬가지예요!
- "어떤 데이터베이스를 쓸지", "어떤 API 키를 쓸지"를 알아야 해요
- 이 정보들을 **환경 변수**라고 부릅니다

### 🎮 Render는 뭐예요?

- Render는 우리 웹사이트를 인터넷에 올려주는 서비스예요
- 마치 "웹사이트 호텔" 같은 곳이에요!
- 우리 웹사이트가 여기서 살면서 작동해요

---

## 📋 단계별 설정 가이드

### 현재 상태 확인 ✅

화면을 보니 이미 **DATABASE_URL**이 추가되어 있네요!
- ✅ DATABASE_URL = PostgreSQL 연결 문자열

이제 나머지 3개만 추가하면 됩니다!

---

### 1단계: YOUTUBE_API_KEY 추가하기

**이게 뭐예요?**
- YouTube에서 데이터를 가져오려면 "열쇠"가 필요해요
- 이게 바로 YouTube API 키예요!

**설정 방법:**

1. 화면 아래에 있는 **"+ Add"** 버튼을 클릭하세요
   - 보통 왼쪽 아래에 있어요

2. 두 개의 빈 칸이 나타날 거예요:
   - **KEY** (왼쪽 칸)
   - **VALUE** (오른쪽 칸)

3. 왼쪽 칸(KEY)에 입력:
   ```
   YOUTUBE_API_KEY
   ```
   - 대소문자를 정확히 입력하세요!

4. 오른쪽 칸(VALUE)에 입력:
   ```
   AIzaSyAQdvDGLrVzHYWz5XNKPEYCvWWJi5ZEnAY
   ```
   - 이건 YouTube에서 받은 열쇠예요!

5. 다른 곳을 클릭하거나 Enter를 누르면 저장돼요!

**✅ 완료!** YOUTUBE_API_KEY가 추가되었습니다!

---

### 2단계: NEXT_PUBLIC_BASE_URL 추가하기

**이게 뭐예요?**
- 우리 웹사이트의 주소예요!
- "우리 집 주소" 같은 거예요

**설정 방법:**

1. 다시 **"+ Add"** 버튼을 클릭하세요

2. 왼쪽 칸(KEY)에 입력:
   ```
   NEXT_PUBLIC_BASE_URL
   ```

3. 오른쪽 칸(VALUE)에 입력:
   ```
   https://realxbest.onrender.com
   ```
   - 이건 Render가 우리에게 준 주소예요!

4. 저장하세요!

**✅ 완료!** NEXT_PUBLIC_BASE_URL이 추가되었습니다!

---

### 3단계: NODE_ENV 추가하기

**이게 뭐예요?**
- "지금은 실제 서비스 중이에요!"라고 알려주는 거예요
- 개발 중인지, 실제 서비스 중인지 구분해요

**설정 방법:**

1. 다시 **"+ Add"** 버튼을 클릭하세요

2. 왼쪽 칸(KEY)에 입력:
   ```
   NODE_ENV
   ```

3. 오른쪽 칸(VALUE)에 입력:
   ```
   production
   ```
   - "production"은 "실제 서비스 중"이라는 뜻이에요!

4. 저장하세요!

**✅ 완료!** NODE_ENV가 추가되었습니다!

---

## 🎉 최종 확인

이제 화면에 4개의 환경 변수가 보여야 해요:

1. ✅ DATABASE_URL = PostgreSQL 연결 문자열
2. ✅ YOUTUBE_API_KEY = AIzaSyAQdvDGLrVzHYWz...
3. ✅ NEXT_PUBLIC_BASE_URL = https://realxbest.onrender.com
4. ✅ NODE_ENV = production

## 🚀 저장하고 배포하기

모든 환경 변수를 추가했으면:

1. 화면 오른쪽 아래를 보세요
2. **"Save, rebuild, and deploy"** 버튼이 보일 거예요
3. 이 버튼을 클릭하세요!
   - "저장하고, 다시 만들고, 배포하세요"라는 뜻이에요

**⏳ 잠깐만요!**
- 배포하는 데 5-10분 정도 걸릴 수 있어요
- 조금만 기다려주세요!

---

## ✅ 완료!

배포가 끝나면:
- ✅ 웹사이트가 작동해요!
- ✅ https://realxbest.onrender.com 에서 확인할 수 있어요!

## 🆘 문제 해결

### "+ Add" 버튼이 안 보여요
- 화면을 아래로 스크롤해보세요
- Environment Variables 섹션 아래에 있어요

### 값이 저장이 안 돼요
- KEY와 VALUE를 모두 입력했는지 확인하세요
- 다른 곳을 클릭하거나 Enter를 눌러보세요

### 실수로 잘못 입력했어요
- 각 변수 오른쪽에 휴지통 아이콘(🗑️)이 있어요
- 클릭하면 삭제할 수 있어요
- 다시 추가하면 돼요!

---

## 💡 팁

- **대소문자 주의**: KEY는 정확히 입력해야 해요!
- **공백 주의**: 앞뒤 공백이 없어야 해요
- **따옴표 없이**: VALUE는 따옴표 없이 입력하세요
