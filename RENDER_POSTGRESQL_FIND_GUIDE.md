# Render에서 PostgreSQL 데이터베이스 찾기 가이드 (초등학생 수준)

## 🎯 목표
Render 웹사이트에서 PostgreSQL 데이터베이스를 찾아서 연결 정보를 복사하는 것입니다.

---

## 📋 단계별 설명

### 1단계: Render 웹사이트 열기

1. **브라우저를 엽니다** (Chrome, Edge, Firefox 등 아무거나)
2. **주소창을 클릭합니다** (맨 위에 있는 긴 박스)
3. **`https://dashboard.render.com`** 을 입력합니다
4. **엔터 키를 누릅니다**
5. **로그인합니다** (이메일과 비밀번호 입력)

---

### 2단계: 대시보드에서 데이터베이스 찾기

#### 방법 A: 왼쪽 메뉴에서 찾기 (가장 쉬운 방법)

1. **왼쪽에 있는 메뉴를 봅니다**
   - 왼쪽에 여러 개의 버튼들이 있습니다
   - "Dashboard", "Services", "Databases" 같은 글자들이 보입니다

2. **"Databases" 또는 "데이터베이스"를 찾습니다**
   - 왼쪽 메뉴에서 "Databases" 글자를 찾습니다
   - 만약 안 보이면 아래로 스크롤해봅니다
   - 또는 "New +" 버튼 옆에 있을 수 있습니다

3. **"Databases"를 클릭합니다**
   - 마우스로 클릭합니다

4. **데이터베이스 목록이 보입니다**
   - 여러 개의 데이터베이스가 보일 수 있습니다
   - 또는 비어있을 수도 있습니다 (아직 만들지 않았다면)

---

#### 방법 B: 상단 메뉴에서 찾기

1. **맨 위에 있는 "New +" 버튼을 찾습니다**
   - 오른쪽 위에 있는 "+" 표시가 있는 버튼입니다

2. **"New +" 버튼을 클릭합니다**
   - 클릭하면 메뉴가 나타납니다

3. **메뉴에서 "PostgreSQL"을 찾습니다**
   - "PostgreSQL" 또는 "Postgres" 글자를 찾습니다
   - 클릭합니다

---

### 3단계: 데이터베이스가 있는지 확인하기

#### ✅ 데이터베이스가 있는 경우:

1. **데이터베이스 목록에서 우리가 사용하는 것을 찾습니다**
   - 이름이 `realxbest-db` 또는 비슷한 이름일 수 있습니다
   - 또는 다른 이름일 수도 있습니다

2. **데이터베이스를 클릭합니다**
   - 클릭하면 상세 페이지가 열립니다

3. **다음 단계로 넘어갑니다** (4단계)

---

#### ❌ 데이터베이스가 없는 경우:

**아직 데이터베이스를 만들지 않았습니다!** 먼저 만들어야 합니다.

**데이터베이스 만드는 방법:**

1. **"New +" 버튼을 클릭합니다**
   - 오른쪽 위에 있는 "+" 버튼입니다

2. **"PostgreSQL"을 클릭합니다**
   - 메뉴에서 "PostgreSQL"을 찾아 클릭합니다

3. **데이터베이스 설정 화면이 나타납니다**
   - 여러 가지 입력 칸이 보입니다

4. **이름 입력하기:**
   - "Name" 또는 "이름" 칸에 `realxbest-db` 입력
   - 또는 원하는 이름 입력

5. **데이터베이스 이름 입력하기:**
   - "Database" 또는 "데이터베이스" 칸에 `realxbest` 입력
   - 또는 원하는 이름 입력

6. **지역 선택하기:**
   - "Region" 또는 "지역" 선택
   - 다른 서비스와 같은 지역 선택 (예: Oregon, Singapore 등)

7. **플랜 선택하기:**
   - "Plan" 또는 "플랜" 선택
   - 처음에는 "Free" 선택 (무료)

8. **"Create Database" 또는 "생성" 버튼 클릭**
   - 맨 아래에 있는 버튼을 클릭합니다

9. **잠시 기다립니다**
   - 데이터베이스가 만들어지는 동안 기다립니다 (1-2분)
   - "Provisioning" 또는 "생성 중" 메시지가 보입니다

10. **완료되면 데이터베이스 상세 페이지가 열립니다**

---

### 4단계: 연결 정보 복사하기

데이터베이스 상세 페이지가 열렸습니다. 이제 연결 정보를 복사해야 합니다.

1. **페이지를 살펴봅니다**
   - 여러 가지 정보가 보입니다
   - "Connection Info" 또는 "연결 정보" 버튼을 찾습니다

2. **"Connection Info" 버튼을 찾습니다**
   - 보통 페이지 중간이나 위쪽에 있습니다
   - 파란색 버튼일 수 있습니다
   - 또는 회색 박스 안에 있을 수 있습니다

3. **"Connection Info"를 클릭합니다**
   - 클릭하면 연결 정보가 나타납니다

4. **"External Connection String"을 찾습니다**
   - 여러 가지 연결 방법이 보입니다
   - "External Connection String" 또는 "외부 연결 문자열"을 찾습니다
   - 보통 가장 아래쪽에 있습니다

5. **"External Connection String" 옆의 복사 버튼을 클릭합니다**
   - 작은 복사 아이콘(📋) 또는 "Copy" 버튼을 클릭합니다
   - 클릭하면 복사됩니다!

6. **복사 확인:**
   - 복사된 값은 이렇게 생겼습니다:
   ```
   postgresql://user:password@host:5432/database
   ```
   - 또는 포트 번호가 없을 수도 있습니다:
   ```
   postgresql://user:password@host/database
   ```

---

### 5단계: GitHub Secrets에 붙여넣기

1. **새 탭을 엽니다**
   - 브라우저에서 새 탭을 엽니다 (Ctrl+T)

2. **GitHub 주소 입력:**
   - 주소창에 다음을 입력합니다:
   ```
   https://github.com/VENCEO86/realxbest/settings/secrets/actions
   ```
   - 엔터 키를 누릅니다

3. **로그인합니다** (필요한 경우)

4. **"DATABASE_URL"을 찾습니다**
   - 목록에서 "DATABASE_URL"을 찾습니다

5. **연필 아이콘(✏️)을 클릭합니다**
   - "DATABASE_URL" 오른쪽에 있는 연필 아이콘을 클릭합니다

6. **기존 값 삭제:**
   - "Value" 칸을 클릭합니다
   - 전체 선택 (Ctrl+A)
   - 삭제 (Delete 키)

7. **복사한 값 붙여넣기:**
   - 붙여넣기 (Ctrl+V)
   - 복사한 연결 문자열이 붙여넣어집니다

8. **포트 번호 확인:**
   - 붙여넣은 값에 `:5432` 같은 포트 번호가 있는지 확인합니다
   - 없어도 괜찮습니다 (자동으로 추가됩니다)

9. **"Update secret" 버튼 클릭**
   - 맨 아래에 있는 버튼을 클릭합니다

10. **완료!**

---

## 🔍 문제 해결

### 문제 1: "Databases" 메뉴가 안 보여요

**해결 방법:**
- 왼쪽 메뉴를 아래로 스크롤해봅니다
- 또는 "New +" 버튼을 클릭하고 "PostgreSQL"을 선택합니다
- 또는 주소창에 `https://dashboard.render.com/databases` 입력

---

### 문제 2: 데이터베이스 목록이 비어있어요

**해결 방법:**
- 데이터베이스를 먼저 만들어야 합니다
- 위의 "데이터베이스 만드는 방법"을 따라하세요

---

### 문제 3: "Connection Info" 버튼이 안 보여요

**해결 방법:**
- 페이지를 아래로 스크롤해봅니다
- 또는 데이터베이스 이름을 클릭해서 상세 페이지로 들어갑니다
- 상세 페이지에서 "Connection Info"를 찾습니다

---

### 문제 4: 복사 버튼이 안 보여요

**해결 방법:**
- "External Connection String" 옆의 텍스트를 직접 선택합니다
- 마우스로 드래그해서 전체 선택합니다
- 복사 (Ctrl+C)
- GitHub Secrets에 붙여넣기 (Ctrl+V)

---

## 📸 참고 이미지 설명

Render 대시보드에서:
- **왼쪽 메뉴**: 여러 메뉴 항목들이 있습니다

- **"Connection Info"**: 연결 정보를 보려면 클릭합니다
