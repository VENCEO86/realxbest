# 🎯 Render 환경 변수 설정 가이드 (초보자용)

## 📍 목표
Render 대시보드에서 누락된 환경 변수 2개를 추가하는 것입니다.

---

## 🚀 단계별 가이드

### 1단계: Render 대시보드 접속하기

1. **웹 브라우저 열기** (Chrome, Edge, Firefox 등 아무거나)
2. **주소창에 입력**: `https://dashboard.render.com`
3. **Enter 키 누르기**
4. **로그인하기** (GitHub, Google 등으로 로그인)

---

### 2단계: 서비스 페이지 찾기

로그인 후 화면이 나오면:

1. **왼쪽 메뉴에서 "Services" 클릭**
   - 또는 상단에 "Services" 탭이 보이면 클릭
   
2. **서비스 목록에서 "realxbest" 찾기**
   - 서비스 이름이 보일 것입니다
   - 클릭해서 들어갑니다

3. **또는 직접 주소로 가기**
   - 주소창에 입력: `https://dashboard.render.com/web/srv-d48p38jipnbc73dkh990`
   - Enter 키 누르기

---

### 3단계: Environment 섹션 찾기

서비스 페이지에 들어가면:

1. **화면 왼쪽에 메뉴가 보입니다:**
   ```
   Overview
   Events
   Logs
   Environment  ← 이걸 클릭!
   Settings
   ```

2. **"Environment" 클릭**
   - 파란색으로 선택되면 성공!

---

### 4단계: 환경 변수 추가하기

Environment 페이지에 들어가면:

#### 현재 보이는 것:
- 화면 상단에 "Environment Variables" 제목
- 그 아래에 테이블이 있습니다:
  ```
  KEY                    VALUE
  NEXT_PUBLIC_BASE_URL   ••••••••
  NODE_ENV               ••••••••
  YOUTUBE_API_KEY        ••••••••
  YOUTUBE_API_KEYS       ••••••••
  ```

#### 추가할 환경 변수 2개:

**첫 번째 환경 변수 추가:**

1. **"Add Environment Variable" 버튼 찾기**
   - 테이블 위쪽 오른쪽에 있을 수 있습니다
   - 또는 "+" 버튼이 있을 수 있습니다
   - 또는 "Add" 버튼이 있을 수 있습니다

2. **버튼 클릭**

3. **KEY 입력란에 입력:**
   ```
   NEXT_PUBLIC_APP_URL
   ```
   - 정확히 이대로 입력 (대소문자 구분 중요!)

4. **VALUE 입력란에 입력:**
   ```
   https://realxbest.com
   ```
   - 정확히 이대로 입력

5. **"Save" 또는 "Add" 버튼 클릭**
   - 저장 완료!

**두 번째 환경 변수 추가:**

1. **다시 "Add Environment Variable" 버튼 클릭**

2. **KEY 입력란에 입력:**
   ```
   NEXT_TELEMETRY_DISABLED
   ```
   - 정확히 이대로 입력

3. **VALUE 입력란에 입력:**
   ```
   1
   ```
   - 숫자 1만 입력

4. **"Save" 또는 "Add" 버튼 클릭**
   - 저장 완료!

---

### 5단계: 확인하기

추가가 완료되면 테이블에 이렇게 보여야 합니다:

```
KEY                        VALUE
NEXT_PUBLIC_BASE_URL       ••••••••
NODE_ENV                   ••••••••
YOUTUBE_API_KEY            ••••••••
YOUTUBE_API_KEYS           ••••••••
NEXT_PUBLIC_APP_URL        ••••••••  ← 새로 추가됨!
NEXT_TELEMETRY_DISABLED    ••••••••  ← 새로 추가됨!
```

총 **6개**의 환경 변수가 보이면 성공입니다!

---

### 6단계: 배포하기 (중요!)

환경 변수를 추가한 후에는 **반드시 재배포**해야 합니다:

1. **왼쪽 메뉴에서 "Events" 클릭**
   - 또는 상단의 "Events" 탭 클릭

2. **"Manual Deploy" 버튼 찾기**
   - 보통 오른쪽 상단에 있습니다
   - 파란색 버튼일 수 있습니다

3. **"Manual Deploy" 클릭**

4. **"Deploy latest commit" 선택**
   - 팝업이 나오면 이 옵션 선택

5. **"Deploy" 또는 "Confirm" 버튼 클릭**

6. **배포 완료 대기**
   - 화면에 배포 진행 상황이 보입니다
   - "Building..." → "Deploying..." → "Live" 순서로 진행됩니다
   - 약 5-7분 정도 걸립니다

## 🆘 문제 해결

### "Add Environment Variable" 버튼이 안 보여요
- 화면을 스크롤해보세요
- 테이블 위쪽을 자세히 보세요
- "Edit" 버튼을 클릭해보세요 (있을 수 있습니다)

### 환경 변수를 잘못 입력했어요
- 환경 변수 옆에 있는 **연필 아이콘(✏️)** 클릭
- 또는 환경 변수를 클릭하면 수정 가능합니다
- 수정 후 "Save" 클릭

### 환경 변수를 삭제하고 싶어요
- 환경 변수 옆에 있는 **휴지통 아이콘(🗑️)** 클릭
- 또는 환경 변수를 선택하고 "Delete" 버튼 클릭

### 배포가 실패했어요
- "Logs" 탭으로 가서 오류 메시지 확인
- 오류 내용을 확인하고 다시 시도
