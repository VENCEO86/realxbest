✅ 해결된 문제
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ 증상:
   - 우측 상단 드롭다운 메뉴의 한글이 깨짐
   - "ì»¤ë®¤ë‹^í‹°" (커뮤니티)
   - "êºëŸ¬ë" (갤러리)
   - 기타 모든 한글 텍스트 깨짐

✅ 원인:
   - HTTP 응답 헤더에 charset=utf-8이 명시되지 않음
   - Apache 기본 charset 설정 비활성화 상태

✅ 해결:
   1. common.php에 UTF-8 헤더 추가
      → header('Content-Type: text/html; charset=utf-8');
   
   2. Apache charset 설정 활성화
      → AddDefaultCharset UTF-8
   
   3. MySQL 문자셋 모드 활성화
      → G5_MYSQL_SET_MODE: true

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 수정된 파일
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. common.php
   - 15번 줄: UTF-8 헤더 추가
   - P3P 헤더 다음에 Content-Type 헤더 설정

2. data/dbconfig.php
   - G5_MYSQL_SET_MODE: true로 변경
   - MySQL UTF-8 모드 활성화

3. Apache 설정
   - /etc/apache2/conf-available/charset.conf
   - AddDefaultCharset UTF-8 활성화
