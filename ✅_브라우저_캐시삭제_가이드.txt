📝 Chrome 브라우저 캐시 삭제 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

방법 1) 단축키 사용 (빠름) ⭐ 추천

  1. Ctrl + Shift + Delete 누르기
  
  2. 다음 항목 모두 체크:
     ✓ 쿠키 및 기타 사이트 데이터
     ✓ 캐시된 이미지 및 파일
  
  3. 기간: "전체 기간" 선택
  
  4. "데이터 삭제" 버튼 클릭
  
  5. 브라우저 완전 종료 후 재시작


방법 2) 하드 리프레시 (간단함)

  1. http://localhost:8080 페이지에서
  
  2. Ctrl + F5 누르기
     (또는 Ctrl + Shift + R)
  
  3. 페이지가 완전히 새로 로드됨
  
  ⚠️ 주의: 이 방법은 임시적이므로
     완전한 해결을 위해서는 방법 1 권장


방법 3) 시크릿 모드 테스트

  1. Ctrl + Shift + N 누르기 (시크릿 창 열기)
  
  2. http://localhost:8080/ 입력
  
  3. 한글이 올바르게 보이는지 확인
  
  ✓ 시크릿 모드에서 정상이면 → 캐시 문제 확정
     → 방법 1로 캐시 삭제 필수!
