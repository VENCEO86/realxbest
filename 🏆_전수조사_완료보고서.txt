✅ 해결한 주요 문제 (총 8개)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. ✅ 한글 인코딩 깨짐 문제
   원인: MySQL 연결 시 charset 미설정
   해결: mysqli_set_charset($link, 'utf8') 추가
   결과: 메뉴, 게시글 모두 완벽 표시

2. ✅ 방문자 기록 중복 에러
   원인: UNIQUE KEY 중복 INSERT
   해결: ON DUPLICATE KEY UPDATE 추가
   결과: 에러 없이 정상 작동

3. ✅ 게시판 페이지 로드 실패
   원인: bo_include_head 빈 값에서 include() 시도
   해결: isset() 체크 추가
   결과: 게시판 정상 표시

4. ✅ 썸네일 생성 에러
   원인: null 데이터에 count() 실행
   해결: null 체크 추가
   결과: 게시글 없을 때도 정상 작동

5. ✅ 메뉴 한글 깨짐 (이중 인코딩)
   원인: Latin1으로 저장된 UTF-8 데이터
   해결: 메뉴 재생성 (UTF-8로)
   결과: 커뮤니티, 갤러리 등 모두 정상

6. ✅ 관리자 로그인 실패
   원인: MySQL 8.0에서 PASSWORD() 함수 제거
   해결: hash('sha256') 사용으로 변경
   결과: 로그인 정상 작동

7. ✅ 게시판 테이블 컬럼 누락
   원인: g5_visit 테이블에 vi_browser 등 컬럼 없음
   해결: ALTER TABLE로 컬럼 추가
   결과: 방문자 기록 정상

8. ✅ 존재하지 않는 게시판 참조
   원인: 템플릿에 하드코딩된 PAY, GNU_TIP 등
   해결: 실제 게시판(gallery, free)으로 변경
   결과: 모든 최신글 정상 표시
