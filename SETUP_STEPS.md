# 🚀 YouTube API 키 발급 가이드 (사용자가 직접 해야 할 작업)

## ⚠️ 중요: 이 작업은 사용자가 직접 해야 합니다

제가 대신할 수 없는 작업입니다. Google 계정 로그인과 API 키 발급이 필요합니다.

---

## 📋 단계별 가이드 (약 5-10분 소요)

### 1단계: Google Cloud Console 접속

1. 브라우저에서 다음 링크 열기:
   ```
   https://console.cloud.google.com/
   ```

2. **Google 계정으로 로그인**
   - Gmail 계정이 있으면 사용
   - 없으면 새로 만들기

### 2단계: 프로젝트 생성

1. 상단 메뉴에서 **"프로젝트 선택"** 클릭
2. **"새 프로젝트"** 클릭
3. 프로젝트 이름 입력 (예: "korxyoutube")
4. **"만들기"** 클릭
5. 프로젝트가 생성될 때까지 대기 (약 30초)

### 3단계: YouTube Data API v3 활성화

1. 왼쪽 메뉴에서 **"API 및 서비스"** > **"라이브러리"** 클릭
2. 검색창에 **"YouTube Data API v3"** 입력
3. **"YouTube Data API v3"** 클릭
4. **"사용"** 버튼 클릭
5. 활성화 완료까지 대기 (약 10초)

### 4단계: API 키 생성

1. 왼쪽 메뉴에서 **"API 및 서비스"** > **"사용자 인증 정보"** 클릭
2. 상단 **"+ 사용자 인증 정보 만들기"** 클릭
3. **"API 키"** 선택
4. API 키가 생성됨 (팝업에 표시)
5. **⚠️ 중요: API 키 복사** (나중에 다시 볼 수 없을 수 있음)

### 5단계: API 키 제한 설정 (선택사항, 보안 권장)

1. 생성된 API 키 옆 **연필 아이콘** 클릭
2. **"API 제한사항"** 탭
3. **"키 제한"** 선택
4. **"YouTube Data API v3"** 체크
5. **"저장"** 클릭

## ✅ 완료 후 작업

API 키를 받으셨으면 다음 명령어를 실행하세요:

```powershell
# Windows PowerShell에서
$env:YOUTUBE_API_KEY="여기에_발급받은_API_키_붙여넣기"
$env:DATABASE_URL="postgresql://user:password@localhost:5432/korxyoutube"

# 데이터베이스 마이그레이션 (데이터베이스가 설정되어 있다면)
npm run db:push

# 실제 채널 데이터 수집
npm run fetch-channels
```

## 🆘 문제 해결

### "프로젝트를 만들 수 없습니다" 오류
- Google Cloud Console에 결제 정보가 등록되어 있어야 할 수 있습니다
- 무료 체험 계정으로도 가능합니다

### "API를 활성화할 수 없습니다" 오류
- 프로젝트가 완전히 생성될 때까지 기다려보세요
- 페이지를 새로고침해보세요

### API 키가 작동하지 않음
- API 키에 따옴표 없이 입력했는지 확인
- YouTube Data API v3가 활성화되었는지 확인
