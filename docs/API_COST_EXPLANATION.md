# YouTube API 비용 설명 (초등학생도 이해하기)

## 🎁 YouTube API는 무료입니다!

### 쉽게 설명하면:

**YouTube는 하루에 10,000개의 "쿠폰"을 무료로 줍니다!**

- 채널 정보를 1번 가져올 때마다 쿠폰 1개를 사용합니다
- 한 번에 최대 50개 채널을 가져올 수 있습니다
- 하루에 10,000개의 쿠폰을 사용할 수 있습니다

## 📊 비용 계산 예시

### 매일 2만개 채널 불러오기:

```
2만개 채널 = 20,000개
50개씩 묶어서 가져오기 = 400번 요청
필요한 쿠폰 = 400개
하루 무료 쿠폰 = 10,000개
남는 쿠폰 = 9,600개 ✅
```

**💰 비용: 0원 (무료!)**

### 더 많은 데이터도 가능해요!

- 하루 최대: 약 50만개 채널까지 무료로 가능
- 2만개는 충분히 무료 범위 안입니다!

## 📈 실제 사용 시나리오

### 시나리오 1: 매일 1,000개씩 추가
- 첫날: 1,000개 저장
- 둘째날: 2,000개 저장 (누적)
- 20일 후: 20,000개 저장
- 매일 불러오기: 20,000개 × 400 쿠폰 = **무료!**

### 시나리오 2: 매일 2만개 불러오기
- 매일 20,000개 채널 정보 업데이트
- 필요한 쿠폰: 400개
- 하루 무료 쿠폰: 10,000개
- **결과: 무료!**

## ⚠️ 주의사항

1. **쿠폰은 매일 자정에 리셋됩니다**
   - 오늘 사용한 쿠폰은 내일 다시 받을 수 있어요

2. **너무 빠르게 요청하면 안 됩니다**
   - 1초에 1번씩 요청하는 것이 안전해요
   - 너무 빠르면 일시적으로 막힐 수 있어요

3. **여러 API 키를 사용하면 더 많이 가능해요**
   - API 키 1개 = 하루 10,000 쿠폰
   - API 키 2개 = 하루 20,000 쿠폰
   - API 키 5개 = 하루 50,000 쿠폰

- 2만개는 무료 범위의 4%만 사용합니다
