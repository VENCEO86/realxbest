# 50개국 지원 가이드

## 현재 지원 국가 수
현재 시스템은 **약 50개국**을 지원합니다.

## 전체 국가 지원이 어려운 이유

### 1. YouTube API 제한사항
- **쿼터 제한**: YouTube Data API는 하루에 제한된 요청 수만 허용
- **배치 처리**: 한 번에 최대 50개 채널만 조회 가능
- **Rate Limiting**: 너무 많은 요청 시 API 차단 가능

### 2. 데이터 수집 비용
- **50개국 × 평균 100개 채널 = 5,000개 채널**
- 각 채널마다 API 호출 필요
- 실시간 업데이트 시 쿼터 소모량 급증

### 3. 성능 문제
- **데이터베이스 크기**: 수만 개 채널 데이터 저장 필요
- **API 응답 시간**: 배치 처리로 인한 지연 (50개씩 처리)
- **메모리 사용량**: 대량 데이터 처리 시 서버 부하

### 4. 국가별 데이터 가용성
- 일부 국가는 YouTube 채널이 적음
- 국가 코드 매핑 불일치 가능성
- 언어/지역 설정이 없는 채널 존재

## 해결 방안

### 단계적 확장 전략
1. **1단계**: 주요 20개국 우선 지원 (현재 진행 중)
2. **2단계**: 30-40개국으로 확장
3. **3단계**: 50개국 전체 지원

### 최적화 방법
1. **캐싱**: 자주 조회되는 국가 데이터 캐싱
2. **점진적 로딩**: 필요한 국가만 선택적으로 로드
3. **백그라운드 업데이트**: 주기적으로 데이터 갱신
4. **데이터베이스 최적화**: 인덱싱 및 파티셔닝

### 모니터링 방법
1. **국가별 채널 수 추적**
2. **API 쿼터 사용량 모니터링**
3. **성능 메트릭 수집**
4. **에러 로그 분석**
