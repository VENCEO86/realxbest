해결책 1) 브라우저 캐시 완전 삭제 (95% 해결)

  1. 모든 브라우저 창을 완전히 닫기
  
  2. 작업 관리자 (Ctrl + Shift + Esc) 열기
  
  3. "프로세스" 탭에서 브라우저 프로세스 모두 종료
     - Chrome
     - msedge
     - firefox
  
  4. 브라우저 다시 열기
  
  5. Ctrl + Shift + Delete
     - "전체 기간" 선택
     - "쿠키 및 기타 사이트 데이터" 체크
     - "캐시된 이미지 및 파일" 체크
     - "데이터 삭제" 클릭
  
  6. 브라우저 완전 종료
  
  7. 브라우저 다시 시작
  
  8. http://localhost:8080/verify_utf8.php 접속


해결책 2) 시크릿 모드로 테스트 (즉시 확인)

  1. Ctrl + Shift + N (시크릿 모드)
  
  2. http://localhost:8080/verify_utf8.php 입력
  
  3. 한글이 올바르게 보이는지 확인
  
  ✓ 시크릿 모드에서 정상이면
     → 브라우저 캐시가 원인
     → 해결책 1 실행 필수


해결책 3) 다른 브라우저로 테스트

  1. Chrome에서 안 되면 Edge 시도
  
  2. Edge에서 안 되면 Firefox 시도
  
  3. 다른 브라우저에서 정상이면
     → 기존 브라우저의 캐시 문제
     → 해결책 1 실행


해결책 4) Windows 콘솔 인코딩 설정

  PowerShell에서:
  ```powershell
  [Console]::OutputEncoding = [System.Text.Encoding]::UTF8
  chcp 65001
  ```
  
  그 후:
  ```powershell
  curl -s http://localhost:8080/verify_utf8.php
  ```
